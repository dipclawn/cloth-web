# 🌀 Cloth Web

**3D旗帜布料物理模拟 · 网页内容像旗帜一样挂在旗杆上**

[![Live Demo](https://img.shields.io/badge/Live-Demo-6366f1)](https://dipclawn.github.io/cloth-web/)
[![Three.js](https://img.shields.io/badge/Three.js-r160-blue)](https://threejs.org/)

旗面左边缘固定在金色旗杆上，右边缘和底部自由飘动。严格Verlet积分物理模拟，重力+风力+阻尼。

</div>

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🏴 **旗帜物理** | 左边缘固定于旗杆，右边缘自由飘动 |
| 🌬️ **风力** | 可开关的实时风力，布料飘动 |
| 🖱️ **拖拽** | 鼠标/触屏拖拽布料任意位置 |
| 🔄 **视角** | SHIFT+拖拽旋转，滚轮缩放 |
| 📄 **网页纹理** | 旗面显示网页内容 |
